# 마이링크 (MyLink) - 사용자 시나리오 (User Scenario)

> 모든 시나리오는 "사용자는 [목적]하기 위해 [행동]한다." 형식에 맞추어 작성되었으며, 발생 가능한 여러 인터랙션 및 관리 상황을 포괄합니다.

---

## 1. 방문자 (Visitor) 시나리오
> 프로필 이미지 및 닉네임 인지와 같은 단순 열람 단계는 생략하고, 기능적 인터랙션을 중심으로 서술합니다.

* 사용자는 기존의 마이링크 프로필 화면을 이탈하지 않고 원하는 목적지(예: 깃허브, 개인 블로그 등) 사이트로 접속하기 위해, 나열된 버튼 중 특정 링크 블록을 클릭하여 새 탭(New Tab) 환경에서 해당 웹페이지를 열람한다.
* 사용자는 페이지 하단에 숨겨진 또 다른 링크 목적지들을 탐색하기 위해, 스마트폰 화면을 상하로 스크롤(스와이프)하여 가려진 링크 버튼들을 확인하고 클릭한다.
* [AI 제안] 사용자는 자신이 발견한 유용한 마이링크 주소 리스트를 다른 사람에게도 전달하기 위해, 페이지 내에 제공되는 **[공유하기]** 버튼(네이티브 공유 창 또는 URL 복사)을 클릭한다.

---

## 2. 소유자 (Owner) 시나리오

### 2.1 가입 및 프로필 관리
* 사용자는 이메일 가입/인증 등의 절차를 생략하고 즉석에서 멀티 링크 서비스를 생성하기 위해, 최초 접속 랜딩 화면의 **[구글 계정 연동 로그인]** 버튼을 클릭한다.
* 사용자는 외부 SNS 바이오나 블로그 프로필에 붙여넣을 내 서비스 고유 주소를 확보하기 위해, 관리자 에디터 화면 최상단에 노출된 `내 링크 주소(URL)` 영역을 복사(Copy)한다.
* 사용자는 타인에게 보여질 자신의 닉네임(활동명)을 매력적으로 표현하기 위해, 상단 프로필 영역의 `username` 텍스트를 클릭하고 별도 모달 없이 인라인 폼에서 즉시 내용을 수정/저장한다.
* 사용자는 활동명 텍스트 아래에 부가적인 설명을 더해주기 위해, 비어있는 텍스트(소개글) 영역을 클릭하여 짧은 타이틀 문구를 인라인으로 작성한다.

### 2.2 신규 링크 추가 및 검증
* 사용자는 자신의 갓 완성된 새로운 포트폴리오를 페이지 리스트에 덧붙이기 위해, 기존 링크 목록 하단 영역의 **[+ 링크 추가]** 버튼을 클릭하여 빈 입력 컴포넌트를 생성해 낸다.
* 사용자는 방금 생성된 빈 링크 블록에서 방문자의 클릭을 효과적으로 유도하기 위해, 기본 할당된 '제목' 텍스트를 직접 누르고 직관적인 홍보 문구(예: "신규 포트폴리오 바로가기")를 덮어씌워 입력한다.
* 사용자는 버튼을 클릭한 방문자가 실제 목적지로 정상 접속되도록 절대 경로를 맵핑하기 위해, 제목 바로 아래의 'URL' 입력란을 클릭하고 링크 경로(`https://...`)를 붙여넣기 한다.
* 사용자는 본인이 입력한 URL의 아이콘이 정상적으로 잘 연동되었는지 시각적으로 신뢰하기 위해, URL 등록 완료 즉시 버튼 블록 좌측에 렌더링된 '구글 API 기반의 파비콘 이미지'를 육안으로 확인한다.

### 2.3 링크 수정 및 삭제
* 사용자는 과거에 등록한 특정 유튜브 영상의 연결 링크가 다른 플레이리스트 주소로 변경되어 이를 반영하기 위해, 기존 등록돼 있던 리스트 블록 내 URL 부분을 클릭하여 새 주소로 인라인 변경 처리한다.
* 사용자는 등록해둔 링크 제목 텍스트에서 띄어쓰기 오류/오타를 발견하고 이를 고치기 위해, 해당 제목 텍스트를 탭(클릭)하여 오타 부분만 지우고 다시 올바르게 입력한다.
* 사용자는 종료된 옛날 이벤트로 트래픽을 유도할 필요가 없어져 링크 더미를 깔끔하게 유지하기 위해, 해당 링크 우측 끝단에 위치한 🗑️(휴지통/삭제) 아이콘을 클릭하여 리스트 데이터를 지워버린다.
* [AI 제안] 사용자는 방금 지우면 절대 안 되는 링크를 데스크탑 마우스 클릭 미스로 우발적 삭제해버린 비상 상황을 정정하기 위해, 삭제 액션 직후 화면 우하단에 팝업처럼 튀어나온 스낵바 알람 내 **[실행 취소(Undo)]** 액션 텍스트를 즉시 클릭하여 해당 링크를 되살린다.

### 2.4 기기 전환 환경 및 접속 제어
* 사용자는 이동 중 대중교통 안에서 모바일 폰으로 급하게 링크 오타를 수정하기 위해, 스마트폰 브라우저에 접속하여 모바일 기기 사이즈에 맞게 반응형으로 재배치된 동일한 에디터 인터페이스 요소들을 터치 조작한다.
* 사용자는 카페나 공용 PC 등 타인의 기기에서 내 프로필 정보 수정을 마친 후 계정 탈취를 방지하고 완전히 편집 모드를 빠져나가기 위해, 우측 상단 메뉴 컴포넌트 내의 **[로그아웃]** 버튼을 직접 찾아 클릭한다.
